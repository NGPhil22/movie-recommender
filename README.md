# Movie Recommender System

An exploratory data analysis and recommendation engine built on 
the MovieLens / TMDB dataset (45,000+ movies).

## Phase 1 — EDA (Complete)
- Rating distributions
- Genre analysis  
- Release trends by decade

## Tech Stack
Python · Pandas · Matplotlib · Seaborn

## Dataset
[The Movies Dataset – Kaggle](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)
